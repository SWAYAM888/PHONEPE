# PHONEPE
This project performs in-depth analysis of PhonePe digital payment data to understand transaction trends, user behavior, and regional performance across India.

The analysis converts raw data into actionable business insights using Python.

🎯 Objective
Analyze state-wise and district-wise transactions
Understand growth trends in digital payments
Evaluate user engagement (Registered Users & App Opens)
Identify popular transaction types
Study device usage patterns
Explore impact of population & density on transactions

🛠️ Tech Stack
Python 🐍
Pandas, NumPy
Matplotlib, Seaborn
Jupyter Notebook / Google Colab
📂 Dataset Description

The dataset contains multiple sheets:

📊 State Transactions & Users
💳 Transaction Type Split
📱 Device Usage Data
🌍 District Transactions
📈 District Demographics
📊 Analysis Performed

✔ Data Cleaning & Preprocessing
✔ Exploratory Data Analysis (EDA)
✔ State-wise Transaction Analysis
✔ Year-wise Growth Trend
✔ Transaction Type Distribution
✔ Device Brand Analysis
✔ Average Transaction Value (ATV)
✔ User Engagement Analysis
✔ District-Level Analysis
✔ Demographic Correlation Study

📈 Key Insights
📌 Certain states show significantly higher transaction volume
📌 Digital payments show consistent year-on-year growth
📌 Positive correlation between population and transaction volume
📌 A few device brands dominate the market
📌 User engagement varies across different regions
